1999-2017: Stromberg LLC/Paychex/Kronos (Lake Mary, FL) - Senior Software Engineer

My chief areas of responsibility were the core "rules engine" and software systems supporting the timeclock farm which involved many types of punch-clock hardware and software. I maintained, troubleshot and added literally dozens of features to this product (too numerous to list).

The work performed involved many different technologies but primarily 2 flavors of Visual Basic (both classic and .Net), Visual C/C++ and Visual C# along with work in Microsoft SQL Server as the database back-end.

I was also engaged by Kronos to assist with software development for another acquisition, Empower Software for whom I wrote the EIHC.Manager and later the ETLD.Manager. These were systems that allowed integration of the Empower Software clock farm with the Kronos DCM and Time-Link-Direct subsystems. These were both WinForms applications developed in C#.
